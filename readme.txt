Displays personalized greeting messages based on the time of day and weather conditions.

== Description ==

This plugin provides a shortcode to display personalized greeting messages with current weather conditions.

== Installation ==

1. Upload `custom-greeting-message` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter your weather API key in the plugin settings.

== Frequently Asked Questions ==

= How do I get an API key? =

You can obtain an API key by signing up at a weather API provider such as OpenWeatherMap or Weatherstack.
